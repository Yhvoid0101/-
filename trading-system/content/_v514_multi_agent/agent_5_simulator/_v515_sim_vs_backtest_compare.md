## 6. 方法论说明

### 6.1 差异率计算
```
diff_rate = |hifi_metric - ideal_metric| / max(|ideal_metric|, epsilon)
composite_diff = sum(weight_i * diff_rate_i)  # 加权平均
```

指标权重:
- 年化收益率: 30%
- 夏普比率: 25%
- 最大回撤: 20%
- 胜率: 15%
- 总收益率: 10%

### 6.2 因素影响权重计算
```
weight_i = |metric_factor_i - metric_all_disabled| / |metric_all_enabled - metric_all_disabled|
```
通过因素隔离实验 (每次只开启一个微观结构因素) 计算各因素对策略表现的独立影响。

### 6.3 微观结构模型
- **点差**: 动态, 基于波动率 (BTC~2bps, ETH~4bps, SOL~8bps, BNB~5bps, XRP~10bps)
- **滑点**: 平方根模型, slippage = base * sqrt(order_size / avg_volume)
- **流动性**: 按交易时段 (亚洲65%/欧洲100%/美国135%)
- **延迟**: Agent-1 WS 实测 (median=222.646ms, p95=1134.866ms)
- **部分成交**: 订单 > 5% 可用流动性时触发
- **资金费率**: 8h 周期 (BTC~0.01%/8h, 山寨~0.02%/8h)
